# 24BCY70087-EXPERIMENT-3
# Full Stack Experiments Dashboard (Experiment 3)

## Aim

To create a single dashboard that integrates multiple experiments and allows users to navigate between them.

## Objective

* Combine multiple experiments into one interface
* Implement navigation between different sections
* Demonstrate UI design, CRUD operations, and OOP concepts
* Build an interactive and responsive dashboard

## Experiments Included

### 🔹 Experiment 3.1 – Product Card

* Displays product details (name, price, stock)
* Toggle stock status (In Stock / Out of Stock)
* Demonstrates UI component design and dynamic behavior

### 🔹 Experiment 3.2 – Library Management

* Add new books
* Delete books
* Search books
* Demonstrates CRUD operations and DOM manipulation

### 🔹 Experiment 3.3 – Person Class Hierarchy

* Base class: Person
* Derived classes: Student, Teacher
* Method overriding and polymorphism
* Displays object data in UI

## Technologies Used

* HTML
* CSS
* JavaScript (ES6)

## Features

* Single-page dashboard
* Clickable navigation cards
* Dynamic UI updates
* Clean and simple design
* All experiments integrated in one file

## How to Run

1. Save the file as `index.html`
2. Open it in any modern browser (Chrome recommended)
3. Click on any experiment card to view functionality

## Learning Outcome

* Understanding front-end development basics
* Applying CRUD operations using JavaScript
* Learning OOP concepts (inheritance, polymorphism)
* Building a structured and interactive UI

## Conclusion

This project demonstrates how multiple concepts like UI design, JavaScript logic, and object-oriented programming can be combined into a single dashboard for better organization and usability.
